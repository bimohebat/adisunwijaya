About me

Hi, I'm Bimo Hadi Wijaya —
Founder of Adisun Wijaya

I build websites and web systems end-to-end — from the first wireframe to a working dashboard that a client can actually use. My background is in practical, real-world builds: student admission platforms, reporting systems, and profile sites for schools and institutions.

Services

What I can build for you

Every project starts from the same question: what does this site need to do for the people using it? From there, I pick the simplest stack that gets it done well.

Institutional & admission systems

Registration forms, admin dashboards, and print-ready reports for schools and organizations — built to be simple for staff to run day-to-day.

Web applications

Custom platforms built on Laravel and MySQL — submission flows, dashboards, exports, and anything in between.

Profile & landing sites

Fast, clean single-page sites for businesses and institutions — built with plain HTML, CSS, and JS, ready to deploy anywhere.

Maintenance & fixes

Debugging, environment setup, and cleanup for existing sites — from local XAMPP quirks to Node and Vite build issues.

Reporting & exports

Auto-generated documents, QR-coded records, and chart-based dashboards that turn raw data into something readable.
